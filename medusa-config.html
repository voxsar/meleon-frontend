Quick Setup Examples

You can also configure via JavaScript console:

// Set configuration
MedusaConfig.configure(
    'https://your-backend.com',
    'pk_your_publishable_key'
);

// Or set individually
MedusaConfig.setBackendUrl('https://your-backend.com');
MedusaConfig.setPublishableKey('pk_your_key');
